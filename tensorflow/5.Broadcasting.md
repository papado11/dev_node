# 5 Broadcasting

```python
tf.constant([
    [0, 0, 0],
    [10, 10, 10],
    [20, 20, 20],
    [30, 30, 30],
]) + tf.constant([0, 1, 2])
# -> 自动扩张为:[[0, 1, 2],[0, 1, 2],[0, 1, 2]]
# 最后结果为
# <tf.Tensor: shape=(4, 3), dtype=int32, numpy=
# array([[ 0,  1,  2],
#        [10, 11, 12],
#        [20, 21, 22],
#        [30, 31, 32]])>
tf.constant([[0], [10], [20], [30]]) + tf.constant([0, 1, 2])
# 自动扩展为
# [[0, 0, 0], [10, 10, 10], [20, 20, 20], [30, 30, 30]] +
# [[0, 1, 2], [0, 1, 2], [0, 1, 2], [0, 1, 2]]
# 最后结果为
# <tf.Tensor: shape=(4, 3), dtype=int32, numpy=
# array([[ 0,  1,  2],
#        [10, 11, 12],
#        [20, 21, 22],
#        [30, 31, 32]])>
```
